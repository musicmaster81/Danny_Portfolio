New and Improved Titanic Classification Project

September 25, 2022

Earlier in my Project Portfolio’s history, I created a Machine Learning algorithm to predict the survivors of the Titanic using datasets from the Kaggle competition by the same name. However, this project was created in the early stages of my Machine Learning “learning” curve (pun intended).

This time, I’m back for a second try with all of the additional knowledge of an employment ready ML Engineer (i.e, feature engineering, cross validation, hyperparameter optimization, etc.). The goal of this project is simple: To score higher than a 76% accuracy in prediction ratings.

Here are some additional questions that I answered to help create better models:

  • How can we go about categorizing categorical variables?
  • How can we choose the best set of hyperparameters for each model using grid search?
  • Can we create new features from previously existing data?

To cut to the chase, I was able to create a random forest model that scored an accuracy rating of about 77.3%. While a minimal improvement, the techniques that I learned for constructing a Machine Learning workflow were invaluable.
